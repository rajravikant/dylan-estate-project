
## Run Locally

Clone the project

```bash
  git clone https://github.com/rajravikant/Dylan-estate-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

for clinet
```bash
  npm run dev 
```


for server
```bash
  npm start

Create a env file and store the keys
